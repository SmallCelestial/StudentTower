## Requirements
Our game is based on pygame library, however additionals libraries are needed.  
Full list of libraries needed is below:  
  
**Pygame** -library responsible for game engine:
```
pip install pygame
```
**sqlite3** -library responsible for database:
```
pip install sqlite
```
